Описание

База данных инвентори предназначена для хранения информации о местонахождении различного оборудования, материалов, основных средств и их учёта.

Программа состоит из графического интерфейса пользователя и базы данных в формате MySQL.

Внешний вид программы

В строке заголовка находятся данные о предприятии, подразделении и сотруднике, работающем с базой данных.

Под строкой заголовка расположено поле фильтра, подробное описание которого приведено в разделе Работа с фильтром.

В центре находится поле для вывода информации.

Внизу - функциональные кнопки. Кнопка "Добавить/Переместить" предназначена для внесения новой информации в базу данных о устройстве или его местонахождении. Кнопка "Редактировать" используется для редактирования уже существующей информации. Поле "Инструменты" содержит различные встроенные функции для обработки информации из базы данных. Кнопка "Отчёт" позволяет экспортировать данные из таблицы в формат *.csv с разделителями точка с запятой для дальнейшей работы с данными, например, в приложении MS Excel.

Добавление нового вендора

Для добавления нового производителя оборудования поле Производитель должно быть пустым и нажмите кнопку Добавить. Появится предупреждение.

Если вы уверены в правильности наименования производителя оборудования и его отсутствия в существующем списке, нажмите "Да" - новый производитель будет добавлен в список и можно переходить к внесению информации об оборудовании.

Инструменты

Кликните правой кнопкой мыши на поле Инструменты, во всплывающем меню будет отображён список доступных инструментов.

Активные пользователи

Выводит список пользователей, работающих в данный момент с программой.

Статистика

Выводит статистическую информацию о базе данных.

Распределение: в работе по ЦТЭ

Отображает таблицу с распределением оборудования, находящегося в работе, по ЦТЭ и участкам. Применяется для расчёта количества несинжаемого запаса.

Подготовка отчётов

Информацию из поля для вывода можно сохранить в виде отчёта в формате *.csv с разделителями точка с запятой для дальнейшей обработки, например в табличном редакторе MS Excel.
